# 📈 Customer Churn Prediction System

An end-to-end Machine Learning project to predict customer churn using XGBoost and Streamlit.

## 🚀 Features
- Customer churn prediction
- Data preprocessing pipeline
- ML model training
- Interactive Streamlit UI
- Business insights

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- XGBoost
- Streamlit

## 📊 Model Performance
- Accuracy: ~79%

## ▶ Run Project

```bash
streamlit run app.py
